gura #

vlang.io | Docs | Changelog | Contributing

>>> import gura { parse, encode }
>>> data := parse('text: "Hello World!"') ?
>>> println(data)
{'text': gura.Any('Hello World!')}
>>> encode(data)
text: "Hello World!"
