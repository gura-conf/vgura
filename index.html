gura #

V Gura

vlang.io | Docs | Changelog | Contributing

>>> import gura { raw_parse, raw_encode }
>>> data := raw_parse('text: "Hello World!"') ?
>>> println(data)
{'text': gura.Any('Hello World!')}
>>> raw_encode(data)
text: "Hello World!"

type Any #

type Any = Null
	| []Any
	| bool
	| byte
	| f32
	| f64
	| i16
	| i64
	| i8
	| int
	| map[string]Any
	| string
	| u16
	| u32
	| u64

Any is a sum type that lists the possible types to be decoded and used.
